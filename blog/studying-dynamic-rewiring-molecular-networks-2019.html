Studying the dynamic rewiring of molecular networks

A comprehensive exploration of how Natural Language Processing, Data Integration and Graph Analysis contribute to understanding molecular network dynamics

Talk Overview

In this talk, I showcase how methods from Natural Language Processing, Data Integration and Graph Analysis all contribute to studying the dynamic rewiring of molecular networks. I first explain some basic concepts from Text mining and show how they can ultimately be used to perform pathway reconstruction from literature, or how to mine clinical trials for structured information.

Further, I illustrate the need for data integration, and apply differential network analyses on a plant osmotic stress use-case. The analysis ultimately revealed 3 novel genes for further experiments, and the results were published in BMC Bioinformatics.

Natural Language Processing

Advanced text mining techniques for extracting meaningful information from biomedical literature and clinical trial documents.

Data Integration

Sophisticated methods for combining heterogeneous biological data sources to create comprehensive molecular networks.

Graph Analysis

Mathematical approaches for analyzing network topology and identifying dynamic changes in molecular interactions.
